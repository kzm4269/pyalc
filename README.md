# PyALC

## Install

```bash
$ pip install git+https://github.com/kzm4269/pyalc.git
```

Using Pipenv:
```bash
$ pipenv install --selective-upgrade "git+https://github.com/kzm4269/pyalc.git#egg=alc"
```
